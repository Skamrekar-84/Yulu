# Yulu - Hypothesis Testing

## Project Overview

This project analyses the Yulu bike-sharing dataset to identify factors affecting bike rental demand using Exploratory Data Analysis (EDA) and Statistical Hypothesis Testing.

The analysis focuses on understanding how variables such as weather, season, temperature, humidity, and working day conditions influence bike rental behaviour.

---

## Objectives

* Perform Exploratory Data Analysis (EDA)
* Understand customer rental behaviour
* Analyse the impact of weather and seasonal conditions
* Apply statistical hypothesis testing
* Generate business insights for demand forecasting

---

## Dataset Features

* Season
* Holiday
* Working Day
* Weather
* Temperature
* Humidity
* Wind Speed
* Casual Users
* Registered Users
* Total Rental Count

---

## Techniques Used

### Exploratory Data Analysis

* Distribution plots
* Boxplots
* Countplots
* Correlation heatmap
* Outlier detection

### Statistical Tests

* Shapiro-Wilk Test
* Levene’s Test
* Independent Sample T-Test
* ANOVA Test
* Chi-Square Test

---

## Key Insights

* Weather significantly impacts bike rental demand.
* Higher humidity reduces rentals.
* Certain seasons show higher bike usage.
* Registered users contribute heavily to total rentals.
* Working day status showed no significant difference in rental count.
